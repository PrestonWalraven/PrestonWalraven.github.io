Data Structures Teaching Assistant Description

Overview

Data Structures (CSCI 262) is a class that defines commonly used data structures, introduces algorithm analysis and teaches about abstract data types, templates, and inheritance in C++.

  • Provided support for 25+ students through office hours and in-class lab guidance
  • Effectively graded and provided feedback for students taking the course
  • Contributed to course direction decisions to improve the quality of lectures

My role as a Teaching Assistant for Data Structures (CSCI 262) consists of:

  • Assisting in Lab sections to enable and promote a beneficial learning experience for students.
  • Grading projects and providing valuable feedback for students to enhance their knowledge on the subject.
  • Providing feedback on coursework and student progress to the instructors in order to improve the class for future semesters.

Type of help I give to students:

  • Technical C++ advice
  • Coding style guidance

Data Structures Class Description:

Defining and using data structures such as linked lists, stacks, queues, binary trees, binary heap, and hash tables. Introduction to algorithm analysis, with emphasis on sorting and search routines. Language skills: abstract data types, templates, and inheritance.
